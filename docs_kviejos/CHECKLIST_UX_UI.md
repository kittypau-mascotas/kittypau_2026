# Checklist UX/UI — KViejos

## P0 (no negociable)
- Mensajes grandes y legibles (TV y movil)
- Alertas con accion sugerida clara
- Estado "hub offline" visible
- No duplicar alertas (dedupe por ventana de tiempo)

## P1
- Personalizacion de horarios de silencio
- Reporte semanal para familia/cuidador

## P2
- Temas accesibilidad (daltonismo, lectura facil)
- Voice prompts (si hardware lo permite)

## Definition of Done (release)
- 1 flujo completo demo (evento -> alerta -> historial)
- Evidencia: screenshot + log (audit_events)
- Prueba en 2 tamanos de pantalla movil y 1 TV target
