# Producto y UX — KViejos

## Vistas principales (familia)
- Inicio: estado de hoy (alertas activas + ultimos eventos)
- Historial: por dia/semana (movimiento, puertas, riesgos)
- Rutinas: medicamentos (horarios) + confirmaciones
- Contactos: quien recibe alertas, canales

## Canal TV (UX primaria para adulto mayor)
Objetivo: avisos claros y no invasivos.
- Overlay superior: recordatorios (medicamento, agua, cita)
- Alertas de riesgo: mensaje destacado + instruccion (ventilar, cerrar llave)
- Confirmacion simple (si existe control remoto): "OK"

## Reglas UX
- Tipografia grande, alto contraste, lenguaje directo
- Un evento = un mensaje (evitar duplicados)
- Reintento con backoff (no spamear)

## Estados de error
- Hub offline
- Sensor sin bateria
- WiFi caido
- Alertas silenciadas por horario
