# Evaluación y métricas

## Métricas ML (si hay labels)
- Precision / Recall / F1
- ROC-AUC (si corresponde)

## Métricas de datos
- Missing rate, frescura, duplicados, clocks inválidos

## Drift / estabilidad
- PSI (Population Stability Index) para cambios de distribución de features
- Monitoreo de tasa de alertas (alert fatigue)
