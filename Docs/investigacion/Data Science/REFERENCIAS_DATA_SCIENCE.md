# Referencias para Data Science, ML y Series Temporales

Este documento reúne libros recomendados como base de estudio para análisis de datos, machine learning y forecasting, especialmente útil para proyectos con sensores, predicción y detección de anomalías.

## 1. Base de Data Science

### `Python for Data Analysis`
- Autor: Wes McKinney
- Enfoque: limpieza, transformación y análisis de datos reales con Python
- Por qué importa: es una base práctica para trabajar con `pandas`, exploración y preparación de datasets
- Ideal si trabajas en Python y necesitas una base sólida antes de pasar a modelos

## 2. Data Science + Machine Learning

### `Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow`
- Autor: Aurélien Géron
- Enfoque: machine learning aplicado con ejemplos prácticos
- Qué aporta: conecta análisis de datos con modelos reales y técnicas modernas de ML
- Útil para pasar de EDA y feature engineering a entrenamiento de modelos

## 3. Series Temporales: mejor equilibrio entre teoría y práctica

### `Forecasting: Principles and Practice`
- Autores: Rob J. Hyndman y George Athanasopoulos
- Enfoque: forecasting desde lo básico hasta modelos aplicados
- Qué aporta: explicación clara de ARIMA, ETS, validación temporal y buenas prácticas
- Muy recomendado para proyectos donde importa predecir comportamiento futuro a partir de sensores

## 4. Serie Temporal más formal y profunda

### `Time Series Analysis: Forecasting and Control`
- Autores: George E. P. Box, Gwilym M. Jenkins, Gregory C. Reinsel y Greta M. Ljung
- Enfoque: teoría clásica y base estadística avanzada de series temporales
- Qué aporta: profundidad matemática y fundamento para métodos modernos
- Ideal cuando quieras subir de nivel en estadística e ingeniería de forecasting

## 5. Enfoque moderno en Python

### `Time Series Forecasting in Python`
- Autor: Francesca Lazzeri
- Enfoque: forecasting moderno con Python
- Qué aporta: enfoque práctico con técnicas de ML y deep learning
- Muy útil si quieres llevar lo aprendido a proyectos reales con sensores e IA

## Ruta recomendada

Si el objetivo es tu proyecto de plato inteligente con IA, la secuencia más útil sería:

1. `Python for Data Analysis`
2. `Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow`
3. `Forecasting: Principles and Practice`
4. `Time Series Forecasting in Python`

## Qué te habilita esta ruta

Con esa progresión vas a poder:

- detectar patrones de consumo en gatos
- hacer predicción de comportamiento
- identificar anomalías
- preparar datasets para modelos supervisados y de series temporales
- construir una base seria para análisis exploratorio y machine learning aplicado

## Nota de uso

Este archivo sirve como lista de referencia de estudio. No reemplaza la documentación del proyecto ni las decisiones de modelado, pero sí ayuda a ordenar el aprendizaje y la implementación.
