# Costos de Producción Estimados para KittyPaw

Este documento detalla las estimaciones de costos de producción por unidad de KittyPaw, considerando dos escenarios de abastecimiento: componentes y fabricación en Chile, y componentes y fabricación en China.

## 1. Escenario de Producción en Chile

### 1.1. Costo de Materiales y Fabricación 3D (por unidad)

| Componente                  | Precio Estimado Chile (CLP) |
| :-------------------------- | :-------------------------- |
| Microcontrolador ESP32      | $8.000                      |
| Sensor de Peso HX711 + Celda de Carga | $7.000                      |
| Sensor de Temperatura y Humedad DHT11 | $2.200                      |
| Sensor de Luz LDR           | $300                        |
| Batería Recargable          | $6.000                      |
| Controlador de Carga        | $8.500                      |
| **Subtotal Electrónica**    | **$32.000**                 |
| Carcasa Impresa 3D (mediana) | $20.000                     |
| **Costo Total Materiales y 3D por Unidad** | **$52.000**                 |

### 1.2. Costos Adicionales por Unidad (Estimación)

| Concepto                    | Precio Estimado Chile (CLP) |
| :-------------------------- | :-------------------------- |
| Mano de Obra de Ensamblaje  | $5.000                      |
| Empaquetado                 | $3.000                      |
| Envío y Logística (nacional) | $4.000                      |
| **Subtotal Costos Adicionales** | **$12.000**                 |

### 1.3. Costo Total de Producción por Unidad (Escenario Chileno)

**Costo Total de Producción por Unidad = $52.000 (Materiales y 3D) + $12.000 (Adicionales) = $64.000 CLP**

### 1.4. Precio de Venta Estimado (con 20% de Ganancia sobre Venta)

**Precio de Venta Estimado = $80.000 CLP**

## 3. Cálculo de Gross Profit (GP) por Unidad

### 3.1. Escenario de Producción en Chile (Costo por Unidad: $64.000 CLP)

*   **Con 20% de Ganancia sobre Venta:**
    *   Precio de Venta: $80.000 CLP
    *   **Gross Profit (GP): $16.000 CLP** ($80.000 - $64.000)

*   **Con 40% de Ganancia sobre Venta:**
    *   Precio de Venta: $106.667 CLP (aproximado)
    *   **Gross Profit (GP): $42.667 CLP** ($106.667 - $64.000)

### 3.2. Escenario de Producción con Proveedores Chinos (Costo por Unidad: $44.895 CLP)

*   **Con 20% de Ganancia sobre Venta:**
    *   Precio de Venta: $56.119 CLP
    *   **Gross Profit (GP): $11.224 CLP** ($56.119 - $44.895)

*   **Con 40% de Ganancia sobre Venta:**
    *   Precio de Venta: $74.825 CLP
    *   **Gross Profit (GP): $29.930 CLP** ($74.825 - $44.895)
